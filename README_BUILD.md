# Сборка для продакшена

Для максимальной скорости загрузки запустите перед деплоем:

```bash
npm install
npm run build:css
```

Это создаст `frontend/css/tailwind.min.css` (~80 KB) вместо загрузки Tailwind CDN (~700 KB JS).

После сборки страницы автоматически используют собранный CSS (если файл существует).
