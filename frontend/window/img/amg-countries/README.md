# Фото для рекламных посадочных (Акции)

Папка для хранения фото стран и городов для страниц акций в Яндекс.Директ.

## Структура

- **Страны**: добавьте 3 фото в `$amg_country_tape` в `promotions.php` (или в подпапки по slug: `turkey/`, `egypt/`)
- **Источники фото**:
  1. **TourVisor** — используется автоматически из кэша туров (фото отелей в стране/регионе)
  2. **Локальные** — папки `img/турция`, `img/египет` и т.д.
  3. **Эта папка** — для своих подборок (только с правом использования)

## Требования для рекламы

- Фото должны соответствовать стране/городу (не использовать чужие локации)
- Рекомендуется: 3 фото страны, 1–2 фото на город/курорт
