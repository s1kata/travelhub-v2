Куда класть фото для стран стран (hero + галерея)
================================================

1) Отдельные страницы frontend/window/countries/*.php
   В каждом файле массив $countryData['images'] — пути вида:
   ../img/ИМЯ_ПАПКИ/файл.jpg
   На диске: frontend/window/img/ИМЯ_ПАПКИ/файл.jpg
   Папки часто на кириллице (турция, египет, ОАЭ, шриланка и т.д.) — имена должны совпадать с путями в PHP.

2) Динамическая страница frontend/window/countries/country.php
   Картинки приходят из API/данных страны; при пустой галерее проверьте API и поля images.

3) Офисы
   Фото офисов: /frontend/window/img/offices/{город}/{slug}/ — см. docs/OFFICES.md

4) Проверка
   Если файл из массива images отсутствует, в шаблоне сработает onerror → запасное фото (Unsplash).
