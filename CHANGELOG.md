# Changelog — Travel Hub v2

## v2.0 (июль 2026)

### Дизайн

- Новая палитра (UI/UX Pro): primary `#1A1A40` (deep navy), secondary `#5DA9A4` (teal), CTA `#FF6B6B` (coral) вместо indigo/orange v1.
- **Тотальная перекраска (v2.0.1)**: все оттенки старой гаммы (indigo, оранжевый, sky-blue) заменены на v2-палитру во всех CSS и PHP-файлах (`frontend/`, `backend/components/`), включая inline-стили и Tailwind-утилиты:
  - indigo (`#4F46E5`, `#6366F1`, …) → navy-шкала (`#1A1A40`, `#3D3D74`, …);
  - оранжевый (`#FF6B00`, `#E55F00`, …) → coral-шкала (`#FF6B6B`, `#F65252`, …);
  - sky/blue (`#0EA5E9`, `#38BDF8`, `#3B82F6`, …) → teal-шкала (`#5DA9A4`, `#79BCB7`, …).
- Автосгенерированный файл `frontend/css/v2-tailwind-overrides.css` (генератор `tools/v2_tw_overrides.js`) перекрашивает все использованные в разметке Tailwind-классы `bg/text/border/from/to/ring-sky|indigo|orange-*` без правки HTML; подключается через `@import` из `v2-theme.css`.
- Шапка: тёмный navy-фон, кнопка «Регистрация» — коралловый градиент, активные пункты меню — teal.
- Футер: фон `#10102E` (deep navy).
- Hero главной: новое фото (бирюзовое море), navy-градиент поверх, новый заголовок «Отпуск мечты — в пару кликов».
- Новая типографика: **Outfit** для заголовков, **Work Sans** для текста (Inter остаётся как fallback).
- Файл темы `frontend/css/v2-theme.css` — переопределяет токены `design-system.css` (`--ds-*`), поэтому тема применяется ко всем страницам без правки их разметки.
- Карточки туров: скруглённые углы 18px, мягкие тени, подъём при наведении, цена крупнее и в акцентном цвете, гео-строка в teal.
- Кнопки: коралловый CTA с подъёмом и свечением при наведении; на странице тура CTA перекрашены в акцент v2.
- Модалка бронирования: плавная анимация появления (spring-easing).
- Навигация: анимированное подчёркивание ссылок в шапке.

### Микровзаимодействия и анимации

- `frontend/js/v2-theme.js`:
  - scroll-reveal: секции и карточки плавно появляются при прокрутке (IntersectionObserver);
  - кнопка «Наверх» после 400px прокрутки;
  - уважает `prefers-reduced-motion` (анимации отключаются).

### Лид-воронка (3–4 клика до оплаты)

1. Главная → поиск или «Горящие туры».
2. Результаты → карточка: кнопка **«Забронировать»** (вместо длинного «Просмотреть параметры тура и забронировать»).
3. Кнопка ведёт на страницу тура с `open_booking=1` — **форма бронирования (имя, телефон) открывается автоматически**.
4. Отправка заявки → сразу оплата картой в том же окне (Т-Банк), без перехода на отдельную страницу.

- Тело карточки по-прежнему кликабельно и ведёт на детальную страницу (для тех, кто хочет сначала посмотреть).
- Блок «Оставить заявку менеджеру» **не изменялся** (по просьбе Никиты).

### Подключение темы

- `backend/components/design_system_head.php` — шрифты v2 + `v2-theme.css` + `v2-theme.js` (страны, услуги, VIP-отели и др.).
- `backend/components/header.php` — тема на всех страницах с общей шапкой (about, contacts, офисы, туры).
- `frontend/index.php`, `frontend/window/tour-detail.php` — прямое подключение (эти страницы грузят CSS сами).

### UX-проход «продакт + пожилой пользователь» (v2.0.2)

Аудит пути «поиск → выдача → тур → бронирование» и упрощение по best practice туристических сайтов:

- **Поисковик на главной**: подписи над полями («Откуда летим», «Куда хотим», «Когда», «Ночей», «Кто едет») — как у Booking/Aviasales; сводки без сокращений: «7–14 ночей», «2 взрослых, 1 ребёнок» вместо «7–14 н.», «2 взр.».
- **Фильтры без жаргона**: «Всё включено / Завтрак + ужин / Только завтрак / Без питания» вместо «AI/HB/BB/RO»; «5 звёзд», «Отлично (8+)», «1-я линия (у моря)».
- **Долгий поиск**: если поиск идёт дольше 35 секунд, в лоадере появляется кнопка «Подобрать тур с менеджером» — пользователь не остаётся один на один с крутящимся колесом.
- **Карточки туров**: подпись «цена за 2 взрослых», кнопка «Выбрать этот тур» вместо безликого «Выбрать».
- **Блок «Как купить тур — 3 простых шага»** на главной: найдите тур → нажмите «Забронировать» → оплатите онлайн.
- **Телефон в шапке**: кликабельный `tel:` +7 (846) 254-16-56 на десктопе и в мобильном меню.
- **Страница тура**: кнопки «Забронировать тур» и «Забронировать и оплатить картой»; подпись «Бесплатно и ни к чему не обязывает»; крупные поля формы (44px+), подсказка «Позвоним только по этой заявке»; исправлен заголовок модалки (было «Cоздание заявки бронирования»).
- **Доступность**: видимый фокус (`:focus-visible` teal 3px), тач-зоны от 44px на мобильных, страховка scroll-reveal — контент показывается принудительно через 2.5–3 с, если IntersectionObserver не сработал.

- Структура страниц и URL (кроме нового параметра `open_booking=1`).
- Блок заявки менеджеру на странице тура.
